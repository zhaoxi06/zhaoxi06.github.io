数组的深复制在开发过程中会经常用到，总结记录

<!--more-->

### 1、用数组的slice()
```
var arr1 = [1,2,3];
var arr2 = arr1.slice(0);
arr1[0]=0;
console.log(arr1);  // [0,2,3]
console.log(arr2);  // [1,2,3]
```

### 2、用数组的concat()
```
var arr1 = [1,2,3];
var arr2 = arr1.concat();
arr1[0]=0;
console.log(arr1);  // [0,2,3]
console.log(arr2);  // [1,2,3]
```

### 3、用ES6扩展运算符
```
var arr1 = [1,2,3];
var arr2 = [...arr1];
arr1[0] = 0;
console.log(arr1);  // [0,2,3]
console.log(arr2);  // [1,2,3]
```

### 4、用JSON.parse(JSON.stringify())
```
var arr1 = [1,2,3];
var arr2 = JSON.parse(JSON.stringify(arr1));
arr1[0] = 0;
console.log(arr1);  // [0,2,3]
console.log(arr2);  // [1,2,3]
```
